Fraud Detection Secure API

A Machine Learning-based Fraud Detection System with integrated cybersecurity features.

Features
- Fraud detection using Random Forest
- Secure REST API (Flask)
- API Key Authentication
- Rate Limiting
- IP Blocking & Intrusion Detection
- Real-time Dashboard

Tech Stack
- Python
- Flask
- Scikit-learn
- SMOTE
- HTML/CSS

How to Run

```bash
pip install -r requirements.txt
